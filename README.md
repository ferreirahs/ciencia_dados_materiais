# Ciência de Dados de Materiais para a descoberta de semicondutores

Este repositório contém dados de simulações computacionais DFT (VASP) e modelos de aprendizado de máquina treinados e utilizados para prever propriedades de materiais cristalinos inorgânicos.

Estes dados são oriundos da minha tese de doutorado e estão separados nos três estudos de caso realizados:

1) **DFT + Síntese de Laboratório:** Perovskitas hibrídas pseudo-2D do tipo Dion-Jacobson
2) **DFT High Throughput:** Perovskitas pseudo-1D - Jakobssonita + Ternários e Binários de fases competidoras
3) **Aprendizado de Máquina:** Compostos binários cristalinos
